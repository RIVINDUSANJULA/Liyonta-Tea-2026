## 📖 About The Project

Liyonta Tea is a premium web application built to showcase the delicate balance between modern web engineering and historical brand storytelling. 

The goal of this project was to create a digital storefront that feels like a vintage 1910s archive, serving as the official online home for Liyonta Tea. Instead of relying on heavy 3D rendering or bloated JavaScript libraries, the aesthetic is achieved entirely through precise typography, strict color palette enforcement, and fluid micro-interactions. The result is a deeply immersive, luxury user experience that maintains perfect Lighthouse scores, instantaneous load times, and flawless mobile responsiveness.

> **🚀 View the Live Project:** [Click here to explore the official Liyonta Tea experience](https://liyontatea.vercel.app/)

## ✨ Key Features

* 🕰️ **The 1910s Archive Lightbox:** A custom-built, full-screen image gallery featuring automatic scroll-locking, sepia/grayscale CSS filtering, and a double-matted frame aesthetic to mimic physical historical archives.
* 🟢 **Live Estate Status (Google Maps API):** Integrates real-time data to display the exact opening hours of the tea factory directly in the navigation bar, complete with local-time fallbacks.
* ✨ **Fluid Micro-interactions:** Powered by **Framer Motion**, featuring scroll-linked progress bars, tumbling leaf animations on hover, and buttery-smooth mobile menu transitions.
* ⚡ **Zero-JS Infinite Marquees:** The certification showcase utilizes pure CSS keyframes for infinite, bidirectional scrolling that pauses on hover, ensuring maximum performance without third-party slider bloat.
* 🎨 **Strict Luxury Palette:** A deeply consistent UI using Charcoal (`#2C2A22`), Cream (`#F5F0E8`), and Elegant Gold (`#C8A84B`), enforced flawlessly across all Tailwind classes.

## 🏗️ Architecture & Performance Engineering

This project was engineered with a strict "performance-first" philosophy:

* **Server Components (RSC):** The core storefront (`app/(home)/page.tsx`) is heavily server-rendered, ensuring blistering fast initial paints and pristine SEO.
* **Client Boundary Isolation:** Interactive elements (like the Navigation bar and the vintage lightbox) are isolated strictly to client components, keeping the initial JavaScript payload incredibly minimal.
* **Font Optimization:** Relies on Next.js native font optimization (`next/font`) for *Cormorant Garamond* and *DM Sans* to ensure zero Cumulative Layout Shift (CLS) on load.
